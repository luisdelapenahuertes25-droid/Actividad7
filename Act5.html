Actividad 5 - Configuración de servicios

Documentación de la instalación y configuración de los servicios WEB y FTP en Raspberry.

Servidor WEB

Para empezar descargaremos net-tools y unzip.

Abrimos el archivo de configuración YAML para asignar una IP fija necesaria para los servicios.

La configuración final queda de la siguiente forma:

Aplicamos cambios de permisos y activamos la configuración de red.

Comprobamos el funcionamiento en localhost desde el cliente.

Acceso desde cliente: http://192.168.1.136

Servidor FTP

Instalamos el servicio vsftpd.

Iniciamos y habilitamos el servicio:

systemctl start vsftpd
systemctl enable vsftpd
systemctl status vsftpd

Realizamos copia de seguridad de los archivos de configuración.

Abrimos la configuración del servidor FTP.

Parámetros configurados:

anonymous_enable=NO
local_enable=YES
write_enable=YES
local_umask=022
chroot_local_user=YES
allow_writeable_chroot=YES

Reiniciamos el servicio.

Creamos el usuario FTP.

Creamos carpeta y asignamos permisos.

Abrimos puertos necesarios en el servidor.

Desde el cliente usamos FileZilla con IP, usuario y puerto configurado.

Transferimos un archivo .txt al servidor FTP.
